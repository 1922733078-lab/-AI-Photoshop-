# GitHub 公开注意事项

本仓库计划公开实验样品、软件操作截图、平台生成图和 PSD 文件。公开前建议确认以下事项：

## 1. 文件体积

- 仓库总大小约 700 MB 以上，首次 push 和 clone 时间较长。
- `delete_object_ps2026_layered.psd` 超过 GitHub 普通单文件限制，已拆分为分卷。
- 不需要 Git LFS 即可上传当前整理后的仓库。

## 2. 软件截图

Photoshop 过程截图包含桌面环境、软件菜单、任务栏图标和操作界面。公开前如需进一步匿名化，可自行裁剪截图。

## 3. 平台条款

仓库包含 Adobe Photoshop、千问、即梦、豆包等工具生成或处理的图像。公开展示和二次使用应遵守相关平台服务条款。

## 4. 学术使用

建议在论文、报告或课程材料中说明：

- 结果反映的是 2026 年 5 月 12 日实验时的具体输出。
- 生成式模型和在线平台可能更新，后续复现实验结果可能不同。
- 本实验重点比较工作流、分辨率保持、可编辑性和二次修改能力，而非建立通用模型排名。
